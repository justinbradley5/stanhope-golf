⛳ One-Time Setup

Stanhope Golf needs a Firebase database so your whole group shares the same data in real time. It's free and takes about 5 minutes.

  1. Go to console.firebase.google.com
  2. Click Add project → name it "Stanhope Golf" → Continue
  3. Disable Google Analytics → Create project
  4. Click Realtime Database in the left menu → Create database
  5. Choose Start in test mode → Enable
  6. Click ⚙️ gear → Project settings → scroll to "Your apps" → click </> (Web)
  7. Register app → copy the firebaseConfig values below
⚠️ Paste ALL fields below. Leave any blank if you don't have it.
